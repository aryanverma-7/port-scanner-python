# 🔍 Port Scanner using Python

## 📌 Description

This project is a multi-threaded TCP port scanner developed using Python. It scans a target system to identify open ports by attempting socket connections.

## 🚀 Features

* Scan custom port ranges
* Multi-threaded scanning for faster performance
* Real-time output display
* Saves results to file
* Error handling for invalid input

## 🛠️ Technologies Used

* Python
* Socket Programming
* Networking (TCP/IP)
* Threading

## ▶️ How to Run

```bash
python scanner.py
```

Enter:

* Target IP
* Port range

## 📷 Example Output

```
[OPEN] Port 80
[OPEN] Port 443

SCAN COMPLETE
Total Open Ports: 2
```
